
# Art Vision

Art vision is a streaming service focused currently on digital art.
## Usage/Examples

### Build the frontend
```bash
cd frontend
npm run build
```

### Run the app
```bash
cd serveApp
go build serveApp/cmd
./cmd.exe
```

### Run the backend
```bash
cd backend
go build artvision/backend/cmd
./cmd.exe
```

# NOTE
Database is mariadb
